# SDK

## 预期特性

主要打包两类SDK：
1. 云API接口。基于requests库打包成方便调用的通用格式，并且处理响应里的异常信息。
2. 算法工具函数。把在特定API接口里用到的算法进行打包，方便开发者使用。
